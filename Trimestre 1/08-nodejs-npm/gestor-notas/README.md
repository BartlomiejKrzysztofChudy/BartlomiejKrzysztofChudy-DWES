# Editor de notas (Tema 6 + NPM)

Este directorio rescata el ejercicio del tema 6 y lo empaqueta como proyecto **npm**.

## Instalación
npm install


## Uso
npm start

1. **Crear nota** → escribe el nombre y el contenido (finaliza con dos líneas vacías consecutivas).
2. **Editar nota** → selecciona una nota existente, revisa su contenido y añade texto adicional (mismo criterio para finalizar).
3. **Eliminar nota** → selecciona la nota que desees borrar.
4. **Salir** → termina la aplicación.
